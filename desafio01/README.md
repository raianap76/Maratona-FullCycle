### Usando docker

##### Executar imagem docker

    $ docker run raianabrito/esquenta-maratona-go

##### Ou criar sua própria imagem

Na pasta do [Dockerfile] execute o comando

\$ docker build -t <nome_imagem> .

Então execute

    $ docker run <nome_imagem>
